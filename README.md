# greetings

## Requisitos

- [Python](https://www.python.org/)
  - Os pacotes [hug](https://pypi.org/project/hug/) e[hug_middleware_cors](https://pypi.org/project/hug_middleware_cors/).
- [NodeJS](https://nodejs.org/)

Para executar a api em Python basta executar o comando `hug -f main.py` que ele vai rodar em localhost na port `8000`.

No caso do front-end basta ter instalado [Yarn](https://classic.yarnpkg.com/) ou [NPM](https://www.npmjs.com/) instalado no seu computador, acessar a pasta front e executar `yarn install` para instalar os pacotes, em seguida, executar o comando `yarn start` para rodar localmente na sua m√°quina.
